# gradle-maven-kotlin-dsl
publish with 'maven-publish',  'bintray' , Kotlin DSL

# usage

set these properties in your project.properties file


```groovy

myBintrayRepo=xxx
myPublishedGroupId=com.xxxx
mySiteUrl=https://xxxx
myGitUrl=https://xxx
myDeveloperId=xxx
myDeveloperName=xxx
myDeveloperEmail=xxx
myLicenseName=xxxx
myLicenseUrl=xxxx
myAllLicenses=xxxx

```
